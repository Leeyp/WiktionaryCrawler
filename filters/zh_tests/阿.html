Stroke order

Han character

This entry lacks etymological information. If you are familiar with the origin of this term, please add it to the page as described here.

阿 (radical 170 阜+5, 8 strokes, cangjie input 弓中一弓口 (NLMNR), four-corner 71220, composition ⿰阝可)

  1. prefix for people's names
  2. used in transliteration for /a/

Descendants

  • ア

References

  • Dai Kanwa Jiten: character 41599
  • Dae Jaweon: page 1850, character 2
  • Hanyu Da Zidian: volume 6, page 4119, character 11
